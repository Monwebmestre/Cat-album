# Album de chat
Etes-vous… chien ou chat ? Vous avez dit... chat !? Ça tombe bien. Voici un diaporama photo de chattes que j’ai développé dans le cadre de mon apprentissage de la programmation en Javascript. Cet album a été créé à partir d’une collection d’images de Pixabay. Dans ce projet d'apprentissage, je me sers de requêtes AJAX pour récupérer les données du fichier JSON de Pixabay. Puis en utilisant du Javascript (largement), mais aussi du CSS et du HTML, je permets à l’utilisateur d’afficher les images et de les faire défiler, en gros. Bref, essayez mon application pour voir... si vous êtes plutôt chien, vous finirez chat (hahaha)
